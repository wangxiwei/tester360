# 容量统计

统计文件/目录容量可以用``du``命令

```
NAME
       du - estimate file space usage

SYNOPSIS
       du [OPTION]... [FILE]...
       du [OPTION]... --files0-from=F

DESCRIPTION
       Summarize disk usage of each FILE, recursively for directories.

       Mandatory arguments to long options are mandatory for short options too.
```

``du``常见用法：

```bash
$ du file
$ du -h file
$ du -sh /path/to/dir
```
